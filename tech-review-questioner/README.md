# Tech Review Questioner

技术方案深度评审与优化助手，模拟"打破砂锅问到底"的领导风格。

## 功能特性

- **多维度评分**：7 个维度独立评分，输出 0-100 分综合报告
- **深度追问**：生成 60+ 个关键追问，覆盖方案全生命周期
- **答案优化**：评估原方案答案，给出优化建议和推荐答案
- **薄弱点诊断**：识别方案短板，按优先级排序
- **修改指导**：提供完整的优化框架和对比表

## 评分维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 背景阐述 | 15% | 量化程度、紧迫性、数据支撑 |
| 方案对比 | 20% | 备选方案、对比深度、选择依据 |
| 技术细节 | 20% | 架构设计、性能指标、技术参数 |
| 实现路径 | 10% | 开发计划、里程碑、人力预估 |
| 成本收益 | 10% | ROI 分析、成本拆解、收益指标 |
| 风险预案 | 15% | 风险识别、回滚方案、监控告警 |
| 文档规范 | 10% | 结构清晰、图表规范、专业度 |

## 使用方式

1. 准备好技术方案文档
2. 加载此 Skill
3. 将技术方案内容提供给 Skill
4. 获取评分报告、追问列表、优化建议

## 输出示例

```
# 技术方案深度评审报告

## 一、综合评分

| 评分维度 | 得分 | 评价 |
|----------|------|------|
| 背景阐述 | 85 | 良好 |
| 方案对比 | 60 | 及格 |
| 技术细节 | 78 | 良好 |
| 实现路径 | 70 | 及格 |
| 成本收益 | 55 | 较差 |
| 风险预案 | 65 | 及格 |
| 文档规范 | 80 | 良好 |
| **综合得分** | **70** | - |

## 二、问题与优化
## 三、薄弱点汇总
## 四、领导追问模拟
## 五、修改建议
```

## 使用场景

- 技术方案评审前自查
- 应对领导追问准备
- 提升方案撰写能力
- 模拟答辩练习

## 注意事项

- 评分结果仅供参考，关键在于理解追问背后的逻辑
- 推荐答案可根据实际情况调整
- 重点关注方案中容易被追问的薄弱点
